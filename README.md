#  Project Overview

The primary objective of this project is to understand how classical quantitative finance models and modern machine learning techniques can be combined to price **American Put Options**.

Unlike European options, American options can be exercised at any time before expiry, making their valuation significantly more complex. Throughout this project, I implemented traditional pricing methods and gradually transitioned towards machine learning-based approximation models.

#  Benchmark Model

The neural network is trained to approximate prices generated by the **Cox-Ross-Rubinstein (CRR) Binomial Tree**, which serves as the benchmark pricing model.

Model performance is evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- Maximum Absolute Error
- Predicted vs Benchmark Comparison
- Financial Sanity Checks
